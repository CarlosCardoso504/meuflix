Sobre o MeuFlix

O MeuFlix é uma plataforma de streaming de vídeos próprios, inspirada visualmente na Netflix, desenvolvida como simulação da infraestrutura de tecnologia de uma startup. Qualquer pessoa pode subir um título, uma capa e o seu próprio vídeo, sem depender de serviços externos como o YouTube.

Tecnologias usadas

  • Front-end: HTML5, CSS3 e JavaScript puro (sem frameworks), com layout responsivo inspirado no estilo Netflix.
  • Back-end: Node.js com o framework Express.js, responsável pela API REST que cadastra e lista os vídeos.
  • Banco de dados: PostgreSQL, armazenando título, descrição e o caminho dos arquivos de cada vídeo.
  • Upload de arquivos: biblioteca Multer, que recebe a imagem de capa e o vídeo enviados pelo formulário.

Como foi feito

O projeto foi construído em três camadas. A camada de apresentação é um site estático (HTML/CSS/JS) que consome uma API. A camada de aplicação é um servidor Node.js/Express que expõe rotas para listar vídeos (GET /api/videos), buscar um vídeo específico (GET /api/videos/:id) e cadastrar um novo vídeo (POST /api/videos), recebendo a imagem e o vídeo via upload multipart. A camada de dados é um banco PostgreSQL, onde cada vídeo cadastrado vira uma linha na tabela videos. Todo o código foi versionado e implantado em um provedor de nuvem, conforme detalhado abaixo.

Infraestrutura utilizada

Para colocar o sistema no ar, simulando a infraestrutura de uma empresa real, foram usados os seguintes componentes:

Servidor

Web Service Linux (container Node.js) hospedado na Render, responsável por rodar a aplicação Express e servir a API e os arquivos estáticos.

DNS

Gerenciado pela Render através do domínio padrão meuflix.onrender.com, com a opção de apontar um domínio próprio via Cloudflare DNS.

Serviço Web

Node.js + Express.js, expondo uma API REST (JSON) e servindo o front-end estático na mesma aplicação.

Banco de Dados

PostgreSQL gerenciado (Render Postgres), armazenando os metadados de cada vídeo publicado na plataforma.

Firewall

Proteção de borda via Cloudflare (WAF e mitigação de DDoS) somada ao firewall de rede interno da própria Render, que isola o serviço da internet pública.

Nuvem

Render.com, uma plataforma de nuvem (PaaS) que executa a aplicação sobre infraestrutura da AWS, oferecendo deploy automático a partir do GitHub.

Inteligência Artificial

O Claude (Anthropic) foi utilizado como assistente de desenvolvimento para gerar a estrutura inicial do código (front-end, back-end e banco de dados) e a documentação do projeto.
